---------------------

Create an index

  PUT name_of_the_index

ex:-

PUT favorite_candy

Index a document 
POST 

Syntax:

POST name_of_the_index/_doc 
{
  "filed": "value"
}

POST favorite_candy/_doc
{
  "first_name": "venkat",
  "candy": "Sour Skittles"
}



PUT favorite_candy/_doc/1
{
  "first_name": "ravi",
  "candy": "Sweet Skittles"
}
